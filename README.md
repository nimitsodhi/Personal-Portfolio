## üì¶ Installation & Setup

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone <YOUR_GIT_URL>
    cd digital-canvas
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
